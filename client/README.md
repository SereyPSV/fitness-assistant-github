# Фитнес ассистент от команды студентов Result University:
https://fitness-assistant-1033c.web.app/

## Стек технологий
### Frontend:
* Vite
* React
* Redux (Redux Toolkit)
* TypeScript
* FSD
* CSS
* Дизайн-система Mantine UI

### Backend:
* Google Firebase (Realtime Database, Storage)

### Тестирование:
* Jest
* Testing Library

## Запуск приложения 
```js
npm run dev
```
## Запуск тестов
```js
npm run test
```
